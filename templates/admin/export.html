Export Static Website

Generate a fully self-contained, offline-ready copy of your website. Every page is rendered to HTML, all images and videos are bundled in, and everything is packaged into a single ZIP file you can send to a client, host anywhere, or open offline — no server required.

What Gets Exported
  • Homepage and all visible custom pages
  • Gallery page with all uploaded images and videos
  • All uploaded media (logos, section images, gallery files)
  • Active theme, colour scheme, fonts, animations
  • Navigation, footer, social links, CTAs
  • All paths rewritten to relative — works offline or on any host
  • A HOW_TO_OPEN.html guide included in the ZIP

The admin panel is not included in the export — it's a static site, not a CMS. Contact forms use mailto: links in the static version.
